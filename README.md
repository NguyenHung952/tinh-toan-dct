# 📖 Giới thiệu

**Máy Tính Biến Đổi Cosine Rời Rạc (DCT)** là một web tool hỗ trợ tính toán và mô phỏng phép biến đổi DCT từ dữ liệu đầu vào.

Ứng dụng hỗ trợ:

- 📈 Tính DCT 1D
- 🖼 Tính DCT 2D cho ma trận ảnh
- 🔢 Nhập dữ liệu ma trận tùy chỉnh
- 📊 Hiển thị kết quả trực quan
- ⚡ Tính toán nhanh theo thời gian thực

DCT là kỹ thuật quan trọng trong:

- JPEG Image Compression
- Xử lý ảnh số
- Xử lý tín hiệu số (DSP)
- Multimedia Encoding
- Truyền thông đa phương tiện

# ⚙️ Chức năng chính

## 📌 DCT 1D
- Tính biến đổi cosine rời rạc 1 chiều
- Nhập vector dữ liệu tùy chỉnh

## 📌 DCT 2D
- Tính DCT trên ma trận
- Hỗ trợ xử lý dữ liệu ảnh

## 📌 Matrix Input
- Nhập ma trận NxN
- Kiểm tra dữ liệu đầu vào

## 📌 Visualization
- Hiển thị kết quả trực quan
- Giao diện realtime
